## Fix Plan (all optional — sorted by ROI)

### [HIGH] [PERF] Fix LCP preload target
**Where:** `index.html:37`
**Why:** we preload `/brand/nb-mark.webp` but the landing hero paints `hero_banner_coaching_center.avif`. Preloading the wrong image → LCP misses the fast-path.
**Fix:** replace the preload href with the actual hero AVIF (with `imagesrcset`/`imagesizes` if responsive). Cost: 1 line. Estimated LCP gain on `/`: **−200 to −400 ms** on 4G.

### [HIGH] [PERF] Prefer AVIF via `<picture>` on landing
**Where:** all `src/pages/*` and `src/components/landing/*` `<img src="...webp">` on landing.
**Why:** AVIF siblings are 40–55% smaller. Currently only WebP is served.
**Fix:** wrap 8 hero images in `<picture><source type="image/avif" srcset=...><source type="image/webp" srcset=...><img ...></picture>`. No new tooling — files already exist. Est. payload saving: **~350 KB** across landing.

### [MEDIUM] [PERF] Collapse chatty `lessons` fetches
**Where:** wherever `useLessons(courseId)` + `useChapters(courseId)` fire independently.
**Why:** query #1 has 13,339 calls / 24 h — mostly bundle re-fetches. `get_course_bundle(_course_id)` RPC already returns chapters + lessons in one round-trip and gates locked video URLs server-side.
**Fix:** migrate the two hook calls in `MyCourseDetail` + `LessonView` to a single `supabase.rpc('get_course_bundle', { _course_id })`. Est. reduction: **~10,000 DB calls/day**, TTI on `/course/:id` **−150 ms**.

### [MEDIUM] [PERF] Landing image weight budget
**Where:** `src/assets/landing/*.webp` files > 100 KB (4 of 8).
**Why:** exceed the 60 KB WebP budget for above-the-fold assets.
**Fix:** re-encode with `cwebp -q 78 -alpha_q 90 -m 6`. If output ≥ current, keep. Also add `loading="lazy"` on all landing images below index-2.
Est. saving: **~120 KB**.

### [LOW] [OBS] Add `docs/perf/` to CI
**Where:** `.github/workflows/lighthouse-ci.yml`.
**Why:** we have `lighthouserc.json` but no snapshot commit; regressions slip.
**Fix:** publish Lighthouse JSON as workflow artifact + fail PR if LCP regresses > 20%.
